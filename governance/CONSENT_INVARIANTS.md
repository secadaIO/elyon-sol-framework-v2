# Consent Invariants

Status: Draft  
Scope: Consent requirements  
Governance: Elyon-Sol Framework v1.0.0-structure  

---

## Consent Requirements

Consent within the Elyon-Sol Framework must be:
- Explicit
- Contextual
- Informed
- Revocable

Implicit or assumed consent is not permitted.

---

## Invariants

1. Consent does not transfer authority
2. Consent does not override safety constraints
3. Consent does not enable autonomy
4. Consent may be withdrawn at any time

---

## Failure Mode

If consent is unclear, expired, or revoked:
- Processing must halt
- Escalation must occur
- No automated outcome is allowed
